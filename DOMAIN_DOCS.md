# HASAD - Domain Documentation

## Core Entities
- **User**: System users with role-based access.
- **Farmer**: Profile of the individual farmer.
- **Farm**: Geographic and operational details of a farm.
- **Damage Report**: Documentation of specific agricultural damage.
- **Attachment**: Photos, videos, and voice notes linked to reports.

## Workflows
1. **Assessment**: Agricultural engineers visit farms to document damage.
2. **Review**: Ministry officials review and approve reports.
3. **Analysis**: FAO/UNDP access aggregated data for aid planning.
